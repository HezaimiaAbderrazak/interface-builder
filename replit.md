# NoteFlow AI

A sleek AI-powered note-taking application with glassmorphism UI, 3D backgrounds, and natural language AI assistance.

## Architecture

- **Frontend**: React 18 + TypeScript + Vite (port 5000 in dev)
- **Backend**: Express.js server (port 3000 in dev)
- **Database**: Replit PostgreSQL via Drizzle ORM
- **Auth**: JWT-based (tokens stored in localStorage, signed with SESSION_SECRET)
- **AI Chat**: OpenAI API (gpt-4o-mini) via `/api/chat` server route

## Project Structure

```
src/          - React frontend
  components/ - UI components (NoteCard, AIChatPanel, etc.)
  pages/      - Auth, Index, NotFound
  store/      - NotesContext (syncs with server API)
  lib/        - api.ts (fetch helpers), auth.ts (JWT auth helpers)
  data/       - mockNotes.ts (Note/Tag types)
server/       - Express backend
  index.ts    - Main server entry
  auth.ts     - JWT auth middleware + helpers
  db.ts       - Drizzle + pg pool
  routes/
    authRoutes.ts  - POST /api/auth/register, /login, GET /me
    notesRoutes.ts - CRUD /api/notes
    chatRoutes.ts  - POST /api/chat (OpenAI streaming)
shared/
  schema.ts   - Drizzle schema (users, notes, tags, note_tags)
```

## Dev Workflow

```
npm run dev:server   # Start Express on :3000
vite --port 5000     # Start Vite on :5000 (proxies /api → :3000)
npm run db:push      # Sync Drizzle schema to database
```

## Secrets Required

- `SESSION_SECRET` — Used to sign JWTs (auto-generated by Replit)
- `DATABASE_URL` — PostgreSQL connection string (auto-provisioned)
- `OPENAI_API_KEY` — Optional: enables the AI chat feature

## Key Features

- Masonry note grid with color-coded notes
- Pin, archive, trash with restore support
- AI chat panel (natural language → create notes, set reminders)
- Tags + AI-tagged notes
- Mind map view
- Visual comfort settings (themes, blue light filter, color blindness modes)
- Keyboard shortcuts: Ctrl+K (command palette), Ctrl+N (new note)
